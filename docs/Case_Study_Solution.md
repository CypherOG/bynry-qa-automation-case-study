# QA Automation Engineering Case Study

## Overview

This repository contains my solution for the Bynry QA Automation Engineering Internship Case Study.

The solution demonstrates:

- Debugging flaky Playwright tests
- Framework design using Page Object Model (POM)
- API and UI integration testing
- Multi-tenant validation
- BrowserStack integration strategy
- CI/CD readiness

This framework is designed to be scalable, maintainable, and suitable for testing a B2B SaaS application.
